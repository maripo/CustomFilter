What is CustomBlocker?

This extension allows you create filters freely. For example....

  • Block tweets posted by specific apps on twitter
  • Block comments which contain unpleasant words or tags on social bookmark sites
  • Remove results on specific domains from Google search results

Simple way to create rules

  1. Visit your favorite site.
  2. Click the icon and open pref dialog.
  3. Click "New Rule for this Site"
  4. Edit site RegEx if necessary
  5. Click "Get XPath" button and select XPath which hits desired elements
  6. Input some keywords
  7. Save the rule and reload the page

What is XPath?

XPath is a language for selecting nodes from XML documents. It works well for picking elements from web pages written in HTML.

If you are not familiar with XPath, this extension helps you to get proper XPath and select specific elements as intended. Click "Get XPath" button and select one of target elements. Some XPath expressions are suggested.

When the mouse cursor is over one of suggested XPaths, elements selected by it are highlighted. Click an option which meets intended elements and you can use the selection when the rule is applied.

Of course, you can write XPath manually.

What does the "RegEx" checkbox on the right hand of the keyword textbox?

There are two type of keywords, "Simple Match" and "Regular Expression". The rule works when the content of an element contains simply "Simple Match" keywords or match "Regular Expression" keyword. Keywords which are input when "RegEx" checkbox is checked, they work as "Regular Expression".

I want hide some elements whether they contain any keyword or not.

When "Block Anyway" is checked, elements selected by XPath of "Elements to Hide" will be hidden, whether keywords are contained or not.

What does the number on the icon mean?

It means how many elements are hidden on the current web page.

This functionality can be disabled by unchecking "Show number of blocked elements on app icon" on the preferences page.

How can I stop the extension temporarily?

There is two ways to pause the extension.

Pause all functions temporarily
Click the icon on the Chrome's menu bar and select radio buttons "ON/OFF". When "OFF" is checked, all rules are stopped.
Pause individual rules temporarily
Click the icon on the Chrome's menu bar or show the preference page. You'll find "ON/OFF" buttons on the right side of the list of rules.
