Site

This property specifies URLs on which the rule works. When you are browsing a web site and the URL matches the expression, the rule will work. For example,

  • "http://www\.example\.com/" matches "http://www.example.com/", "http://www.example.com/search?word=xxxxx" etc.
  • "http://www\.example\.com/user/[0-9]+/" matches "http://www.example.com/user/12345/", "http://www.example.com/user/23456/" etc.
